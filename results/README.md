# Results

This folder contains the key outputs generated during the research.

## Structure

results/
├── figures/
│   ├── synthetic/
│   └── hoaxy/
└── tables/
    ├── synthetic/
    └── hoaxy/

## Figures

### Synthetic

This folder contains the main visualizations generated by the synthetic competitive percolation model, including:

- network structure
- spreading dynamics
- strategy comparison
- phase diagrams
- bridge leakage analysis
- fact-check delay analysis

### Hoaxy

This folder contains the figures generated during empirical validation on the Hoaxy dataset, including:

- claim vs fact-check dynamics
- bridge vs intra-community analysis
- k-core analysis
- empirical percolation threshold
- targeted vs random intervention comparison
- temporal centrality analysis

## Tables

### Synthetic

Numerical results from the synthetic simulations, including summary statistics, confidence intervals, critical thresholds, and sensitivity analyses.

### Hoaxy

Numerical results from the empirical validation, including graph statistics, bridge analysis, odds ratios, percolation threshold estimation, strategy comparison, and temporal validation.

Only the most representative figures and tables are stored here to keep the repository lightweight while preserving reproducibility and presentation quality.
